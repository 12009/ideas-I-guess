## 编程技巧

### 求余数

- n%4 == 0
- 可以写成  (n & 3) == 0
- n%8 == 7
- (n & 7) == 7

### 位移运算
- <<      :     左移运算符，num << 1,相当于num乘以2

  \>>      :     右移运算符，num >> 1,相当于num除以2

  \>>>    :     无符号右移，忽略符号位，空位都以0补齐
